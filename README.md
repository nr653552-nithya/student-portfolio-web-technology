📌 Project Title

Student Portfolio Website

📖 Project Description

The Student Portfolio Website is a personal portfolio designed to showcase a student's profile, skills, projects, and contact information. It is developed using HTML, CSS, and JavaScript to create a responsive and user-friendly web application.

This portfolio helps visitors learn about the student's educational background, technical skills, projects, and achievements in an organized manner.
